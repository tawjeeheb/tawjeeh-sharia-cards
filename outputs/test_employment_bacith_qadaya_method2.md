جهات التوظيف وطريقة التقديم

مسار باحث قضايا يظهر في الجهات التي لديها قضايا، نزاعات، لجان، مخالفات، مطالبات، تمثيل أمام جهات قضائية أو شبه قضائية، أو إدارات قانونية متخصصة. والمسميات قد تظهر بصيغ قريبة مثل: باحث قضايا، باحث قانوني، مساعد قضائي، أخصائي قانوني مرتبط بالتقاضي.

القطاع الحكومي

* الجهات العدلية والقضائية (ديوان المظالم، وزارة العدل، المحاكم، اللجان القضائية وشبه القضائية): التقديم عبر جدارات أو بوابات الجهات العدلية الرسمية؛ تعد من أقرب المساحات لهذا المسار بسبب ارتباطها المباشر بالقضايا والملفات العدلية والبحث النظامي.

* الجهات التنظيمية والرقابية ذات اللجان والمخالفات (هيئة الزكاة والضريبة والجمارك، هيئة المحتوى المحلي والمشتريات الحكومية، الهيئة السعودية للملكية الفكرية، الهيئة العامة للمنافسة): التقديم عبر بوابات التوظيف الرسمية أو جدارات؛ تتابع عند وجود لجان نظر مخالفات أو اعتراضات أو منازعات تنظيمية أو مهام مرتبطة ببناء ملفات القضايا.

* الوزارات والهيئات ذات الإدارات القانونية أو القضائية (وزارة التجارة، الهيئة العامة للأوقاف، الهيئة العامة للإحصاء، وزارة الصحة، وزارة النقل والخدمات اللوجستية): التقديم عبر بوابة الوزارة أو الهيئة أو جدارات؛ تتابع عند ظهور مهام مرتبطة بدراسة القضايا أو إعداد الردود أو متابعة النزاعات أو التمثيل أمام الجهات القضائية.

* الجهات الحكومية ذات المشاريع أو الخدمات الكبرى (الهيئة الملكية للجبيل وينبع، الخدمات الصحية بوزارة الدفاع، المكتب الاستراتيجي لتطوير منطقة جازان، المؤسسة العامة لجسر الملك فهد): التقديم عبر بوابات الجهات الرسمية؛ تظهر الحاجة عند وجود عقود أو مطالبات أو نزاعات تشغيلية أو ملفات قضائية مرتبطة بالخدمات والمشاريع.

* إمارات المناطق وأمانات المناطق (إمارة منطقة الرياض، أمانة منطقة الرياض، إمارات المناطق، أمانات المناطق): التقديم عبر جدارات أو القنوات الرسمية عند فتح الإعلان؛ تتابع عند وجود احتياج قانوني أو قضايا أو لجان مخالفات أو مطالبات بلدية وإدارية.

* المراكز القانونية والبحثية الحكومية (مركز الدراسات والبحوث القانونية، مراكز البحوث العدلية، مراكز الدراسات النظامية، الإدارات البحثية القانونية): التقديم عبر بوابات التوظيف الرسمية؛ تناسب من يميل للبحث القانوني وإعداد الدراسات المرتبطة بالقضاء والأنظمة.

القطاع شبه الحكومي

* شركات المشاريع الكبرى والكيانات شبه الحكومية (NEOM، شركات صندوق الاستثمارات العامة، شركات المشاريع الكبرى، شركات التطوير الوطني): التقديم عبر بوابة التوظيف الخاصة بكل جهة عند فتح الإعلان؛ تتابع عند وجود عقود ومطالبات ونزاعات مشاريع أو مهام تمثيل قانوني.

* الهيئات والصناديق والبرامج التنموية (برنامج كفالة، صندوق التنمية الوطني، الصناديق التنموية، البرامج التمويلية): التقديم عبر القنوات الرسمية المعلنة؛ تتابع عند وجود مطالبات تمويلية أو نزاعات عقود أو قضايا ضمانات أو التزامات.

* الجهات ذات العقود والنزاعات والمطالبات الكبيرة (الشركات التنموية، شركات إدارة المشاريع، شركات التشغيل الكبرى، كيانات البنية التحتية): التقديم عبر بوابات التوظيف الرسمية؛ تتابع عند ظهور مهام مرتبطة بالمطالبات أو النزاعات التعاقدية أو متابعة الدعاوى.

القطاع الخاص

* مكاتب المحاماة والاستشارات القانونية (مكاتب المحاماة السعودية، شركات الاستشارات القانونية، مكاتب التقاضي، مكاتب التحكيم): المتابعة عبر مواقع المكاتب أو البريد المهني المعلن؛ تناسب المسار عندما تكون المهام إعداد ملفات القضايا أو البحث النظامي أو تلخيص الأحكام أو دعم فرق التقاضي.

* الشركات ذات إدارات النزاعات والمطالبات والتقاضي (البنوك، شركات التمويل، شركات التأمين، شركات التطوير العقاري الكبرى، شركات المقاولات والمشاريع): المتابعة عبر بوابات التوظيف الرسمية؛ تظهر الحاجة لهذا المسار عند وجود قضايا عملاء أو مطالبات مالية أو نزاعات عقود أو ملفات تقاضٍ.

* الشركات ذات العقود التشغيلية الكبيرة (شركات الاتصالات، شركات الطاقة، شركات الخدمات اللوجستية، شركات التشغيل والصيانة، شركات الخدمات الكبرى): المتابعة عبر بوابات التوظيف الرسمية؛ تتابع عند ظهور مهام مرتبطة بالمطالبات أو المخالفات أو النزاعات التعاقدية أو متابعة الدعاوى.
