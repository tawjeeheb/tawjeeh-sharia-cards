# تقرير تحقق دعم الشهادات المهنية — بطاقة 014 أخصائي تطوير أنظمة ولوائح وسياسات

**تاريخ التحقق:** 2026-06-14 (جولة خامسة — تطبيق الخيار A النهائي)
**المنهجية:** CERTIFICATION SUPPORT REGISTRY v1.0 + WebSearch متعدد
**القائمة المعتمدة:** GRCP + PMP + CFE + CAPM (الخيار A)

---

## ملاحظة منهجية هامة

hrdf.org.sa يُعيد HTTP 403 من بيئة Claude Code السحابية لجميع صفحاته.
القرار التشغيلي المعتمد: تُستخدم المصادر الثانوية الموثوقة وفق CERTIFICATION SUPPORT VALIDATION LOCK v1.0.
حالة GRCP في PDF = (غير مدعومة) بوصفها الحالة المحافظة وفق نظام البطاقة — راجع تفاصيل GRCP أدناه.

## الشهادة الأولى — GRCP

| البند | التفاصيل |
|-------|----------|
| **الاسم الكامل** | GRC Professional |
| **الاختصار** | GRCP |
| **الجهة المانحة** | OCEG |
| **الرابط الرسمي** | https://www.oceg.org/certifications/grc-professional-certification/ |
| **حالة الدعم في PDF** | (غير مدعومة) — الحالة المحافظة لعدم التأكيد الرسمي |
| **حالة الدعم الفعلية** | ⚠️ تحتاج تحقق يدوي — مؤشرات قوية للدعم |
| **سجل الدعم (Registry)** | ⚠️ تحتاج تحقق يدوي |

### مؤشرات الدعم المتاحة
* مقتطف WebSearch: تعويض محدد بـ 7,300 ريال من هدف لشهادة GRCP
* الأكاديمية المالية (fa.gov.sa — جهة حكومية) تقدم برنامجًا تحضيريًا رسميًا: https://fa.gov.sa/Services/ProgramDetails/c95d7b6d-15e9-445c-a2c7-ae6100a0689b

### قرار مطلوب من المستخدم
تحقق من: https://www.hrdf.org.sa/products-and-services/programs/individuals/training/professional-certificates/
ابحث عن: GRCP أو GRC Professional أو OCEG
إذا أُكّدت → حدّث البطاقة إلى (مدعومة) وحدّث هذا الملف.
