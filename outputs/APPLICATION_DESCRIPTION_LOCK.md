# APPLICATION_DESCRIPTION_LOCK
# قفل أسلوب كتابة "طريقة التقديم"

---

## القاعدة الذهبية

طريقة التقديم ليست وصفًا عامًا للجهة.

هي:

> **خريطة دخول للمهنة.**

---

## الترتيب الإلزامي — 4 عناصر متسلسلة

كل فقرة طريقة تقديم تُبنى بهذا التسلسل الثابت:

```
1. نقطة البداية
   (من أين يبدأ القارئ؟ — المنصة أو القناة الأولى)
   ↓
2. قناة التقديم أو المتابعة الرسمية
   (أين يُقدّم أو يُتابع؟ — الرابط أو المنصة أو الموقع المثبت)
   ↓
3. مراحل الترشيح أو المفاضلة أو الاختبار
   (ما الخطوات التالية؟ — فقط إذا كانت مثبتة)
   ↓
4. الملاحظة الحاكمة
   (هل يوجد شرط نظامي أو قيد غير ظاهر؟ — فقط إذا كانت موجودة)
```

---

## الصيغة التحريرية المعتمدة

**الافتتاح الإلزامي:** تبدأ فقرة طريقة التقديم بـ:

> «التقديم يبدأ عبر...» أو «متابعة المسار تبدأ عبر...»

لا تبدأ بـ: «تم إنشاء الجهة...» أو «الجهة تختص بـ...» أو «يمكن للراغبين...»

---

## أمثلة معتمدة

**مسار عام (إعلانات دورية):**

```
وزارة المالية، وزارة التجارة:

التقديم يبدأ عبر منصة جدارة عند فتح الإعلان الرسمي، مع استكمال الملف
الشخصي ومتابعة البوابات الوظيفية للجهات، وتتم المفاضلة وفق الضوابط
المعلنة لكل وظيفة.
```

**مسار سيادي (ترشيح لا إعلان):**

```
المجلس الأعلى للقضاء:

متابعة المسار القضائي تبدأ عبر القنوات الرسمية للمجلس عند تفعيل مسار
الترشيح، ثم تُستكمل إجراءات التحقق من المؤهل واختبار قياس القدرات
والمقابلة وفق قواعد اختيار القضاة المنشورة.
```

**مسار تعاقد مستقل (شركات / مؤسسات):**

```
أرامكو السعودية، سابك:

التقديم يبدأ عبر بوابة Careers الرسمية لكل شركة بإنشاء حساب ورفع
السيرة الذاتية، وتجري الشركة المفاضلة داخليًا دون اشتراط منصة حكومية.
```

## قائمة المحظورات

| العبارة المحظورة | السبب | البديل |
|---|---|---|
| «التقديم عبر الموقع» | غامضة — أي موقع؟ | حدد المنصة أو الرابط |
| «راجع الجهة» | لا تعلّم القارئ شيئًا | اذكر القناة المثبتة |
| «حسب الاحتياج» | لا تحدد متى أو كيف | اذكر آلية المتابعة |
| «يمكن للراغبين...» | يُفضي إلى غموض المسار | ابدأ بنقطة البداية |
| «تم تأسيس الجهة سنة...» | معلومة تاريخية لا تخدم الدخول | احذف — ابدأ بالقناة |
| «تختص الجهة بـ...» | وصف وظيفي لا خريطة دخول | احذف — ابدأ بالمسار |

## منع التفاوت التحريري

إذا استُخدمت في بطاقات مختلفة أساليب متباينة لنفس المعنى:

| الأسلوب المتفاوت | الأسلوب الموحد |
|---|---|
| «يتم التقديم بواسطة...» | «التقديم يبدأ عبر...» |
| «يمكن للراغبين التقديم...» | «التقديم يبدأ عبر...» |
| «تقبل الجهة الطلبات عبر...» | «التقديم يبدأ عبر...» |
| «متابعة المسار عبر...» | يُقبل — لأنه يخدم المسار السيادي |

---

## اختبار الجودة — إلزامي بعد كل فقرة

اسأل بعد كتابة طريقة التقديم:

| السؤال | الإجابة المطلوبة |
|---|---|
| هل يعرف القارئ من أين يبدأ؟ | نعم — نقطة البداية موجودة |
| هل يعرف أين يتابع؟ | نعم — القناة أو المنصة مذكورة |
| هل يعرف هل توجد مراحل ترشيح أو مفاضلة؟ | نعم إذا كانت مثبتة / لا تُذكر إذا لم تثبت |
| هل توجد ملاحظة نظامية مهمة؟ | مذكورة إذا وُجدت / غائبة إذا لم توجد |

- إذا أجبت **نعم** على الأسئلة المنطبقة → الفقرة معتمدة
- إذا أجبت **لا** على أي سؤال منطبق → أعد الصياغة
