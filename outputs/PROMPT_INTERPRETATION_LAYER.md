## الاستثناءات المعتمدة

### 1. عنصر "الشروط والمؤهلات"

**مرجع التنفيذ:** البرومبت المعتمد لهذا العنصر.

**التعديل الوحيد المعتمد:** حذف/منع فقرة "شروط التخصصات الدقيقة" — لا تُدرج.

**البناء الداخلي الإلزامي (بالترتيب):**

1. الشروط العامة
2. شروط المؤهل
3. متطلب العمر
4. الشروط الإضافية

**ممنوع:** إضافة شروط تخصصات دقيقة داخل هذا العنصر تحت أي مسمى.

### 3. عنصر "الدورات الداعمة"

**مرجع التنفيذ:** البرومبت المعتمد لهذا العنصر.

**التعديل الوحيد المعتمد:** تحديث قائمة المنصات المعتمدة.

**المنصات المعتمدة حاليًا (فقط):**

| المنصة | نوع الرابط |
|--------|-----------|
| دروب (Doroob) | `lms.doroob.sa/courses/...` |
| Coursera | `coursera.org/learn/slug` |
| إثرائي | `ethrai.sa/course-details/...` |

**قواعد ثابتة:**

| القاعدة | الحكم |
|---------|-------|
| عدد الدورات | 5 دائمًا |
| لغة اسم الدورة | عربية دائمًا |
| موضع الرابط | على اسم الدورة فقط |
| اسم المنصة داخل اسم الدورة | ممنوع |
| السبب بعد النقطتين | لا يتجاوز 5 كلمات |

**الشكل الإلزامي:**
```
* دورة [اسم الدورة](رابط): سبب لا يتجاوز خمس كلمات
```

### 4. جدول "مدى قبول التخصصات"

**مرجع التنفيذ:** البرومبت المعتمد لهذا العنصر.

**التعديل الوحيد المعتمد:** حذف عمود "ملاحظات دقيقة".

**أعمدة الجدول الإلزامية (بالترتيب):**

| # | اسم العمود |
|---|-----------|
| 1 | التخصص |
| 2 | مدى القبول |
| 3 | السبب |
| 4 | ما يرفع القبول / متطلبات رفع القبول |

**ممنوع:** إضافة أي عمود خامس أو تغيير ترتيب الأعمدة.
