# قفل دائم – LINK VALIDATION LOCK

**تاريخ الاعتماد:** 2026-06-09
**نوع القفل:** إلزامي نهائي — يسري على جميع روابط مشروع Tawjeeh HUB بلا استثناء

---

## العناصر المشمولة

- الدورات الداعمة
- الشهادات المهنية الاحترافية
- برامج التأهيل المعتمدة
- جهات التوظيف وطريقة التقديم
- أي رابط خارجي داخل البطاقات

---

## قاعدة القبول

لا يُسمح بإدراج أي رابط إلا إذا كان:

1. رابطًا نهائيًا مباشرًا
2. يفتح الصفحة المستهدفة مباشرة
3. لا يعتمد على Session أو Login أو Cookies
4. يعمل في نافذة خاصة Incognito
5. لا يحتوي على Redirect مؤقت
6. لا يحتوي على Tracking Parameters
7. لا يُعيد أيًا من الأخطاء التالية:

```
404 / 403 / 500 / ERR_CONNECTION_RESET / Access Denied / Sign In Required
```

## روابط الدورات والشهادات

### ممنوع

```
✗ روابط البحث
✗ روابط الكتالوج العام
✗ روابط التصفية
✗ روابط النتائج
✗ روابط التسجيل المؤقتة
```

### المقبول فقط

الصفحة الرسمية للدورة أو الشهادة نفسها — مثال Coursera:

```
https://www.coursera.org/learn/slug-name
https://www.coursera.org/professional-certificates/slug-name
https://www.coursera.org/specializations/slug-name
```

بدون أي `?` أو `&` أو query parameters.

## روابط الجهات

### ممنوع

```
✗ الصفحة الرئيسية إذا كانت صفحة التوظيف موجودة
✗ صفحات الأخبار
✗ صفحات الإعلان المؤقت إذا كانت صفحة التقديم الدائمة متاحة
```

### المقبول

```
✓ صفحة Careers
✓ صفحة Jobs
✓ صفحة التقديم المباشر
✓ Talent Community
✓ Submit CV
```

## اختبار إلزامي قبل الإخراج

كل رابط يجب أن يحقق **جميع** الشروط التالية:

| الشرط | المطلوب |
|-------|---------|
| يفتح مباشرة | ✓ |
| يعرض المحتوى المقصود | ✓ |
| يعمل بدون تسجيل دخول | ✓ |
| يعمل في نافذة خاصة Incognito | ✓ |
| لا يعتمد على جلسة مستخدم | ✓ |

إذا فشل أي شرط → **يُستبعد فورًا**.

---

## قاعدة المشروع

> **أي معلومة صحيحة مع رابط خاطئ = معلومة مرفوضة.**
>
> الرابط جزء من هوية المعلومة وليس ملحقًا لها.
>
> أي رابط غير مُتحقق منه يُعتبر **خطأً إنتاجيًا جسيمًا** ويُمنع إدخاله في أي ملف مستقبلي.
