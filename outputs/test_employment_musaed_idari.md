1. مساعد إداري

جهات التوظيف وطريقة التقديم

* **الجهات الحكومية المدنية** (الوزارات والهيئات والمصالح الحكومية): التقديم يبدأ عبر منصة جدارات (jadarat.sa) عند فتح الإعلان الرسمي، باستكمال الملف الشخصي ورفع المستندات المطلوبة، وتتم المفاضلة وفق ضوابط كل جهة وإعلانها.
* **أرامكو السعودية، سابك، stc، صندوق الاستثمارات العامة**: التقديم يبدأ عبر بوابة Careers الرسمية لكل جهة مباشرةً بإنشاء حساب وتقديم الطلب — careers.aramco.com / jobs.sabic.com / careers.stc.com.sa / pif.gov.sa/ar/careers — دون الحاجة إلى منصة وسيطة.
* **شركات القطاع الخاص**: التقديم يبدأ عبر منصات التوظيف (LinkedIn، Bayt.com، صبّار) أو عبر صفحة Careers في موقع الشركة مباشرةً إن وُجدت — يُوصى بمراجعة الموقع الرسمي للشركة أولاً للتحقق من قناة تقديم خاصة قبل اللجوء للمنصات العامة.
* **جهات القطاع غير الربحي** (الجمعيات والمؤسسات الأهلية): لا توجد قناة موحدة مثبتة — التقديم يبدأ بمتابعة صفحة الجهة المستهدفة على LinkedIn أو موقعها الرسمي مباشرةً؛ بعض الجهات الكبرى تُعلن عبر جدارات أو موقعها المستقل.
