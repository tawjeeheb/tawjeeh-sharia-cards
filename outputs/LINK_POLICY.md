# سياسة الروابط — Tawjeeh Cards

## المرجع
هذه السياسة مستخلصة من قواعد قفل الروابط المعتمدة في بطاقة «قاضي» (commit `59992bb`).
تسري على جميع البطاقات المهنية الحالية والمستقبلية.

---

## القاعدة الحاكمة

**العنصر المهني مقدّم على سهولة الرابط.**

لا يُستبدل برنامج أو شهادة أو دورة مناسبة مهنيًا لمجرد صعوبة رابطها.
يُثبّت العنصر المهني الصحيح أولًا، ثم يُبحث عن أفضل رابط رسمي مباشر له.
لا يُستبدل العنصر إلا بموافقة صريحة من المنتج المسؤول.

---

## معايير الرابط المقبول

أي رابط داخل أي بطاقة يجب أن يستوفي جميع المعايير التالية:

| المعيار | التفصيل |
|---------|---------|
| رسمي | صادر عن الجهة المالكة للبرنامج/الشهادة/الدورة مباشرة |
| مباشر | يقود إلى صفحة العنصر نفسه، لا إلى الصفحة الرئيسية |
| نشط | يعمل فعليًا ولا يعطي 404 أو redirect إلى صفحة غير ذات صلة |
| محدد | مرتبط بالاسم المذكور في البطاقة، لا باسم مختلف أو أعم |
| قابل للتقديم | يحتوي مسار تسجيل/تقديم/التحاق للبرامج والشهادات والدورات |

## نوع الرابط المطلوب حسب العنصر

### برامج التأهيل المعتمدة
- صفحة البرنامج المباشرة على موقع الجهة المانحة
- أو صفحة التقديم/القبول الخاصة بالبرنامج (إذا كانت منفصلة وواضحة)
- ❌ لا: بوابة قبول عامة كرابط على اسم البرنامج

### الشهادات المهنية الاحترافية
- صفحة الشهادة مع مسار التسجيل/الحجز/الاعتماد
- أو صفحة "How to Earn" / "Apply" / "Register" الرسمية
- ❌ لا: الصفحة الرئيسية للجهة المانحة

### الدورات الداعمة
- صفحة الدورة نفسها مع خيار Enroll / Start / Join
- ❌ لا: صفحة مسار دراسي عام أو قائمة دورات

### جهات التوظيف
- بوابة الوظائف / Careers / Join Us الرسمية للجهة
- ❌ لا: الصفحة الرئيسية للجهة

## روابط ممنوعة منعًا باتًا

```
✗ صفحة رئيسية للجهة (https://example.com/)
✗ صفحة بحث (https://example.com/search?q=...)
✗ صفحة أخبار أو إعلان قديم
✗ صفحة قائمة برامج (إذا وُجدت صفحة البرنامج ذاته)
✗ رابط مختصر من طرف ثالث (bit.ly، goo.gl، إلخ)
✗ رابط تقديري مبني من اسم الجهة دون التحقق
✗ رابط غير رسمي (ويكيبيديا، مدونة، موقع إخباري)
✗ رابط من منصة تدريب طرف ثالث إذا وُجد رابط الجهة الأصلية
```

## شكل الروابط داخل PDF — مقفول

الشكل المعتمد (commit `fe23847`، مُطبَّق في `convert.py`):

| العنصر | القيمة |
|--------|--------|
| لون الرابط | `#023663` (لون المحتوى نفسه) |
| حجم الرابط | `inherit` (حجم النص العادي) |
| الخط السفلي | منقّط `dotted`، سماكة 0.3mm، إزاحة 0.65mm |
| السهم | `↗` داخل نطاق الرابط، بعد آخر كلمة من النص المميز |
| النقطتان الرأسيتان | خارج الرابط |
| نطاق التسطير | من أول اسم البرنامج/الشهادة إلى ما قبل النقطتين مباشرة |

**لا يُعدَّل هذا الشكل إلا بقرار صريح من المنتج المسؤول.**

التطبيق التقني في `convert.py`:
- CSS: `.section-body a.linked-inline-label` و `.section-body .linked-inline-arrow`
- دالة post-processing: `enhance_program_links()` تعمل تلقائيًا على جميع البطاقات

|---------|--------|-------|--------|-----------|--------|-------|-------|---------|

### قيم "الحكم" المسموحة
- `مقبول — صفحة برنامج مباشرة`
- `مقبول — صفحة شهادة مع مسار تسجيل`
- `مقبول — صفحة دورة مع Enroll`
- `تعذر التحقق من داخل البيئة — يحتاج مراجعة يدوية`
- `يحتاج تحديثًا — رابط عام`
- `يحتاج تحديثًا — رابط مكسور`

## منهجية البحث عن الرابط المباشر

إذا لم يُعثر على رابط مباشر بسهولة، اتبع بهذا الترتيب:

1. `site:domain.com "اسم البرنامج"`
2. `site:domain.com enroll OR registration OR apply OR admission`
3. فحص صفحات الجهة: Apply / Register / Admission / Certification portal
4. البحث بالاسم الإنجليزي الكامل إن وجد
5. فحص بوابة التدريب أو التعليم الإلكتروني التابعة للجهة

إذا تعذر إيجاد رابط مباشر بعد البحث العميق:
- أبقِ العنصر المهني، ولا تستبدله
- اكتب في التقرير: "تعذر إيجاد رابط مباشر — يحتاج مراجعة يدوية"
- لا تستخدم رابطًا عامًا بديلًا إلا كحل مؤقت موثّق
