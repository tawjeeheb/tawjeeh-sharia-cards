# برومبت العنصر: جهات التوظيف وطرق التقديم

## هدف العنصر

إنتاج خريطة توظيف عملية وغنية تكشف للقارئ:

* أين توجد الوظيفة فعليًا داخل السوق السعودي.
* ما المسميات التي تمارس داخل كل قطاع.
* ما أنواع الجهات التي توظّف هذه المهنة.
* أمثلة فعلية لجهات مثبتة.
* طريقة تقديم واقعية ومفصّلة لكل نوع جهة.
* قنوات التقديم الحقيقية، لا عبارات عامة.

---

## القاعدة الحاكمة

لا تُذكر أي جهة إلا إذا ثبت أحد الآتي:

* إعلان توظيف سابق أو موثّق.
* وجود إدارة أو وحدة تمارس جوهر المهنة.
* ممارسة تشغيلية مثبتة.
* منتج أو عملية لا تعمل دون هذا الدور.
* سبب بنيوي قوي وقابل للدفاع.

ممنوع إدراج جهة لمجرد شهرتها أو التشابه اللفظي أو إكمال العدد.

---

## ترتيب القطاعات

**إلزامي بهذا الترتيب:**

1. القطاع الحكومي
2. القطاع شبه الحكومي
3. القطاع الخاص
4. القطاع غير الربحي
5. القطاع المستقل

**قاعدة الإدراج:**

* لا يظهر قطاع إلا إذا ثبتت الممارسة الفعلية.
* إذا لم تثبت الممارسة، يُحذف القطاع كاملًا — لا ملاحظات، لا نص نفي.
* لا تُضاف قطاعات لإكمال القائمة.

**قاعدة القطاع المستقل خصوصًا:**

* لا يُذكر إلا إذا كانت المهنة تُمارس فعليًا كخدمة مستقلة أو عمل حر قابل للتسليم.
* إذا كانت الممارسة تتم داخل مكتب أو شركة → هذا قطاع خاص، لا مستقل.
* ممنوع كتابة: "يمكن لاحقًا العمل مستقلًا" أو "بعد الخبرة يمكن...".
* أمثلة مهن تُمارس مستقلًا: مدقق لغوي، مترجم، مصمم، كاتب محتوى.
* أمثلة مهن لا تُمارس مستقلًا: محقق إداري، أخصائي تحقيق قضائي، كاتب ضبط، موظف امتثال داخلي.

## شكل عنوان القطاع

**الصيغة الإلزامية:**

```
القطاع الحكومي (مسمى 1، مسمى 2، مسمى 3)
```

**القواعد:**

* بجانب كل عنوان قطاع، تُكتب المسميات التي تُمارس فعليًا داخل هذا القطاع، بين قوسين.
* المسميات مأخوذة من عنصر "المسميات المكافئة" فقط — لا تخترع مسمى جديدًا.
* لا تضع كل المسميات في كل القطاعات — ضع فقط ما يُمارس فعلًا.
* إذا كان المسمى مناسبًا لأكثر من قطاع، يمكن تكراره بشرط وجود ممارسة واقعية.
* إذا كان ثمة مسمى ضروري غير موجود في "المسميات المكافئة"، أضفه أولًا إلى ذلك العنصر، ثم استخدمه هنا.
* لا تذكر قطاعًا إذا لم يكن له مسمى مناسب أو ممارسة مثبتة.

**مثال:**

```
القطاع الحكومي (محقق قضائي، محقق نظامي، أخصائي تحقيق أمني، شرطي تحري، محقق عمالي)

القطاع شبه الحكومي (محقق نظامي، محقق إداري، أخصائي تحقيق أمني)

القطاع الخاص (محقق إداري، محقق عمالي)
```

## شكل الإخراج النهائي

### ما لا يظهر في الإخراج

لا تستخدم هذه الفئات الداخلية كعناوين في النص النهائي:
* جهات سبق وأعلنت
* جهات توظف لوجود ممارسة/وحدات
* جهات مستحدثة/ناشئة

هذه أدوات بحث داخلية فقط — الإخراج النهائي: **عنوان القطاع مع المسميات + نقاط مباشرة**.

### القالب الإلزامي لكل نقطة

```
* [نوع الجهة أو نطاق الممارسة] (مثل: [جهة1](رابط)، [جهة2](رابط)، [جهة3](رابط)[، جهة4][، جهة5]): [طريقة التقديم الواقعية المفصّلة].
```

| الجزء | المطلوب |
|-------|---------|
| نوع الجهة أو نطاق الممارسة | جملة وصفية تحدد طبيعة الجهات في هذه النقطة — لا تبدأ بأسماء مباشرة |
| (مثل: ...) | أمثلة داخل القوسين — حد أدنى 3، مستهدف 5 عند توفر الدليل |
| طريقة التقديم | قناة أساسية + قناة ثانوية + آلية واضحة — قابلة للتنفيذ فورًا |

---

## عدد النقاط داخل كل قطاع

| الحالة | العدد |
|--------|-------|
| ممارسة واسعة ومتعددة الزوايا | 5 نقاط |
| ممارسة متوسطة | 3–4 نقاط |
| ممارسة محدودة مثبتة | 1–2 نقطة |

كل نقطة يجب أن تُضيف زاوية مختلفة — ممنوع تكرار نفس المعنى.

---

## عدد الأمثلة داخل القوسين

| الحالة | العدد |
|--------|-------|
| توفرت أدلة كافية | 5 أمثلة |
| توفرت أدلة جزئية | 3 أمثلة |
| لم تثبت جهات كافية | اكتفِ بالمثبت ولا تُتمّ العدد شكلًا |

إذا لم يكن بالإمكان تسمية جهة محددة، استخدم تصنيفًا واضحًا — ممنوع اختراع أسماء.

## قواعد طريقة التقديم

### تنويع القنوات حسب طبيعة الجهة

| نوع الجهة | القناة المناسبة |
|-----------|----------------|
| الوزارات | جدارة أو بوابة الوزارة عند الإعلان |
| الهيئات الحكومية | بوابة الهيئة أو إعلان رسمي |
| الجامعات | بوابة التوظيف الجامعية أو جدارة |
| الشركات الكبرى | صفحة Careers أو LinkedIn |
| المكاتب المهنية | البريد المهني أو LinkedIn أو نموذج التواصل |
| الجمعيات | موقع الجهة أو البريد الرسمي |
| المشاريع التشغيلية | شركات التشغيل أو إعلان المشروع |
| العمل الحر | منصات العمل الحر أو التعاقد المباشر |

### قاعدة "عند توفر الإعلان"

إذا كانت الجهة لا تملك تقديمًا مفتوحًا دائمًا، يجب التصريح بذلك:

* "عند توفر الإعلان"
* "عند فتح باب التقديم"
* "عند الإعلان الرسمي عن الشواغر"

ممنوع الإيحاء بأن التقديم مفتوح دائمًا إذا لم يكن كذلك.

### تنويع إلزامي

لا تجعل كل النقاط تنتهي بنفس العبارة. يجب أن تتغير القنوات وأسلوب الوصول من نقطة إلى أخرى.

## قواعد الروابط

* الرابط داخل اسم الجهة أو المنصة فقط — لا روابط خام.
* روابط صفحات Careers مُفضَّلة على الصفحة الرئيسية إذا كانت ثابتة.
* إذا لم تكن صفحة Careers ثابتة، استخدم الصفحة الرئيسية للجهة.

---

## الممنوعات الصارمة

* ممنوع استخدام فئات "سبق وأعلنت / توظف لوجود وحدات / مستحدثة/ناشئة" كعناوين في الإخراج.
* ممنوع كتابة مسميات في عنوان القطاع من خارج "المسميات المكافئة".
* ممنوع إدراج جهة بلا دليل أو سبب بنيوي.
* ممنوع كتابة ملاحظات مهنية داخل هذا العنصر.
* ممنوع تكرار نفس الأمثلة في أكثر من نقطة داخل نفس القطاع.
* ممنوع خلط القطاع الخاص بالقطاع المستقل.
* ممنوع استخدام "قد" أو "ربما" أو "غالبًا".
* ممنوع تعبئة قطاع بملاحظات عامة لا يوجد فيه ممارسة فعلية.
* ممنوع تغيير القالب أو CSS.
* ممنوع تغيير ترتيب عناصر البطاقة.

## اختبار الجودة قبل التسليم

* [ ] عنوان كل قطاع يحتوي المسميات المناسبة بين قوسين.
* [ ] جميع المسميات في عناوين القطاعات مأخوذة من "المسميات المكافئة".
* [ ] لا تظهر فئات "سبق وأعلنت / توظف لوجود وحدات / مستحدثة" كعناوين.
* [ ] كل نقطة تحتوي: وصف النوع + (مثل: ...) + طريقة التقديم.
* [ ] عدد الأمثلة ≥ 3 في كل نقطة عند توفر الدليل.
* [ ] طرق التقديم متنوعة وغير مكررة.
* [ ] الروابط داخل أسماء الجهات والمنصات فقط.
* [ ] القطاعات مرتبة: حكومي → شبه حكومي → خاص → غير ربحي → مستقل.
* [ ] القطاعات غير المثبتة محذوفة كاملًا.
* [ ] القطاع المستقل غائب إذا لم تثبت الممارسة المستقلة.
* [ ] المسميات المكافئة محدّثة إذا احتاج الأمر.
* [ ] يمكن الدفاع عن كل جهة أمام خبير سوق.

إذا فشل أي شرط، أعد البناء قبل الإخراج.

---

## قواعد الإخراج

* لا جداول.
* لا مقدمة.
* لا خاتمة.
* لا شرح للتحليل البنيوي داخل البطاقة.
* الإخراج: عنوان القطاع مع المسميات + نقاط مباشرة.
