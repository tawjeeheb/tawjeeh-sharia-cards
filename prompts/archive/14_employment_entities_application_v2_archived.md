## وظيفة العنصر

عنصر "جهات التوظيف وطرق التقديم" يبني خريطة عملية لجهات التوظيف الفعلية للمهنة داخل السعودية، ويحدد القنوات الواقعية التي يستطيع القارئ استخدامها للوصول إلى الفرصة.

هذا العنصر لا يكتب توقعات عامة، ولا يسرد أسماء جهات شهيرة بلا دليل، ولا يملأ القطاعات شكليًا.

---

## القاعدة الحاكمة

لا تُذكر أي جهة إلا إذا ثبت أحد الآتي:

* إعلان توظيف سابق.
* وجود إدارة أو وحدة تمارس جوهر المهنة.
* ممارسة تشغيلية مثبتة.
* منتج أو خدمة أو عملية لا تعمل دون هذا الدور.
* سبب بنيوي قوي وقابل للدفاع يجعل التوظيف مرجحًا.

ممنوع إدراج جهة لمجرد شهرتها.
ممنوع إدراج جهة لمجرد التشابه اللفظي.
ممنوع إدراج جهة لإكمال العدد.
ممنوع التعميم المؤسسي دون أمثلة مسماة.

---

## ترتيب القطاعات

حكومي → شبه حكومي → خاص → غير ربحي → مستقل

لا يظهر أي قطاع إلا إذا ثبتت ممارسة المهنة فيه فعليًا.

---

## الفئات الثلاث داخل كل قطاع

داخل كل قطاع ثبتت فيه ممارسة المهنة، يتم تصنيف الجهات إلى ثلاث فئات:

### الفئة الأولى — جهات سبق وأعلنت

الجهات التي ثبت لها إعلان فعلي أو وصف وظيفي أو خبر استقطاب موثق لنفس المهنة أو مسمى مكافئ يطابق جوهر الدور.

### الفئة الثانية — جهات توظف لوجود ممارسة/وحدات

الجهات التي لم يظهر لها إعلان موثق، لكن ثبت وجود وحدة أو إدارة أو منتج أو عملية تشغيلية تمارس جوهر المهنة وتحتاجها فعليًا.

### الفئة الثالثة — جهات مستحدثة/ناشئة

الجهات الأحدث أو الأسرع نموًا أو الأقل ازدحامًا بالمنافسة، بشرط وجود سبب بنيوي قوي يربطها بالمهنة.

إذا لم تثبت فئة داخل قطاع، تُحذف ولا تُكتب عبارة نفي.

---

## الشكل الإلزامي للإخراج

### قالب كل سطر

```
* [اسم الفئة]: [وصف نوع/تصنيف الجهات] (مثال: [جهة1](رابط)، [جهة2](رابط)، [جهة3](رابط)[، جهة4](رابط)[، جهة5](رابط)): [طريقة التقديم العملية].
```

**شرح القالب:**

| الجزء | المطلوب |
|-------|---------|
| اسم الفئة | أحد: جهات سبق وأعلنت / جهات توظف لوجود ممارسة/وحدات / جهات مستحدثة/ناشئة |
| وصف نوع/تصنيف الجهات | جملة قصيرة تصف نوع الجهة وسبب ارتباطها بالمهنة — لا تبدأ بأسماء مباشرة |
| (مثال: ...) | أمثلة جهات محددة بروابط — الحد الأدنى 3، المستهدف 5 حيث يوجد دليل |
| طريقة التقديم | القناة الأساسية + القناة الثانوية إن وجدت — قابلة للتنفيذ فورًا |

**قواعد ثابتة للقالب:**

* كل فئة = سطر واحد فقط.
* الأمثلة داخل `(مثال: ...)` دائمًا — لا تخرج منها.
* الرابط داخل اسم الجهة فقط — لا روابط خام.
* طريقة التقديم بعد النقطتين الأخيرتين.
* لا تُكرر طريقة تقديم مطابقة في فئتين متتاليتين بنفس القطاع دون إضافة.

### مثال تطبيقي

```
القطاع الحكومي

جهات سبق وأعلنت
* جهات سبق وأعلنت: جهات قضائية وأمنية وعمالية أعلنت عن وظائف تحقيق رسمية (مثال: [الجهة1](رابط)، [الجهة2](رابط)، [الجهة3](رابط)، [الجهة4](رابط)، [الجهة5](رابط)): التقديم عبر [منصة جدارة](رابط) أو بوابة الجهة عند الإعلان.

جهات توظف لوجود ممارسة/وحدات
* جهات توظف لوجود ممارسة/وحدات: وزارات وهيئات لديها إدارات تفتيش وتحقيق داخلي (مثال: [الجهة1](رابط)، [الجهة2](رابط)، [الجهة3](رابط)): التقديم عبر بوابات الجهات الرسمية مع متابعة المسميات المكافئة.

جهات مستحدثة/ناشئة
* جهات مستحدثة/ناشئة: هيئات تنظيمية حديثة في مراحل بناء وحدات الرقابة (مثال: [الجهة1](رابط)، [الجهة2](رابط)، [الجهة3](رابط)): التقديم المبكر عند فتح الشواغر عبر [منصة جدارة](رابط).
```

## قاعدة الحد الأدنى والمستهدف للأمثلة

| الحالة | العدد |
|--------|-------|
| الحد الأدنى الإلزامي | 3 جهات مسماة |
| المستهدف حيث يوجد دليل | 5 جهات مسماة |
| إذا لم يوجد دليل كافٍ | اكتب الجهات المثبتة فقط ولا تُتمّ العدد شكلًا |

## قواعد الروابط

* الرابط على اسم الجهة أو المنصة فقط — لا روابط خام.
* لا تُذكر منصة تقديم بلا تحقق من وجودها.
* روابط صفحات Careers/Jobs مُفضَّلة على الصفحة الرئيسية إذا كانت ثابتة وقابلة للتحقق.
* إذا لم تكن صفحة الوظائف ثابتة، استخدم الصفحة الرئيسية للجهة.

---

## الممنوعات الصارمة

* ممنوع إدراج جهة بلا دليل أو سبب بنيوي.
* ممنوع إدراج جهة في فئة "سبق وأعلنت" بلا إعلان موثق.
* ممنوع وصف نوع الجهة بعبارات مبهمة مثل: "جهات حكومية كبرى" أو "شركات معروفة" دون تصنيف أدق.
* ممنوع إخراج القالب بدون جزء الوصف قبل القوسين.
* ممنوع إخراج أسماء الجهات خارج القوسين.
* ممنوع تكرار نفس الجهة في أكثر من فئة داخل نفس القطاع.
* ممنوع حذف القطاع الخاص أو المستقل دون بحث.
* ممنوع استخدام "قد" أو "ربما" أو "غالبًا".

## اختبار الجودة قبل التسليم

قبل الإخراج النهائي، تحقق داخليًا من:

* [ ] هل كل سطر يحتوي على: وصف النوع + (مثال: ...) + طريقة التقديم؟
* [ ] هل عدد الأمثلة ≥ 3 في كل فئة؟
* [ ] هل الروابط على أسماء الجهات لا خام؟
* [ ] هل تم ذكر وصف النوع قبل الأمثلة في كل سطر؟
* [ ] هل القطاعات مرتبة: حكومي → شبه حكومي → خاص → غير ربحي → مستقل؟
* [ ] هل تم حذف الفئات غير المثبتة بدل تعبئتها؟
* [ ] هل يمكن الدفاع عن كل جهة أمام خبير سوق؟

إذا فشل أي شرط، أعد البناء قبل الإخراج.
