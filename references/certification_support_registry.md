# سجل دعم الشهادات المهنية الاحترافية — مرجع مشروع بطاقات تجويه

**تاريخ الإنشاء:** 2026-06-14
**مصدر الدعم الرسمي:** برنامج دعم الشهادات المهنية الاحترافية — هدف (hrdf.org.sa)
**ملاحظة بيئية:** hrdf.org.sa يُعيد HTTP 403 من بيئة Claude Code السحابية. جميع أحكام "مدعومة" مستندة إلى مصادر ثانوية موثوقة وليس الوصول المباشر.

---

## قاعدة الاستخدام

عند بناء أي بطاقة جديدة:
1. ارجع أولًا إلى هذا السجل قبل البحث عن دعم الشهادات.
2. إذا كانت الشهادة موجودة بحالة مثبتة → استخدمها مباشرة.
3. إذا كانت غير موجودة → أضفها بعد البحث.
4. لا تُدرج في البطاقة شهادة "تحتاج تحقق يدوي" إلا إذا وافق المستخدم أو لم يوجد بديل مثبت.

---

## رموز حالة الدعم

| الرمز | المعنى |
|-------|--------|
| ✅ مدعومة | ظهر اسم الشهادة أو اختصارها في مصدر موثوق يؤكد دعم هدف |
| ❌ غير مدعومة | تحقق فعلي من غياب الشهادة في قوائم الدعم |
| ⚠️ تحتاج تحقق يدوي | تعذر الوصول للمصدر الرسمي أو الأدلة غير حاسمة |

## تعليمات التحديث

* عند التحقق اليدوي من أي شهادة → حدّث حالتها وأضف المصدر وتاريخ التحقق.
* عند إضافة شهادة جديدة → أضف صفًا جديدًا للجدول.
* إذا تغيرت قائمة هدف → راجع الشهادات بحالة ⚠️ أولًا.
