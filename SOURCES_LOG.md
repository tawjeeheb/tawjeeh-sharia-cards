# سجل المصادر الداخلي

هذا الملف مخصص لتوثيق المصادر التي تُستخدم أثناء بناء عناصر البطاقات.

المصادر ليست عنصرًا ظاهرًا داخل البطاقة النهائية.
تُستخدم فقط للتحقق والمراجعة وربط كل معلومة بدليلها.
